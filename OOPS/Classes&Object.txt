1.What is a Class?
  A class is a blueprint/template.

  Example:
  Before building a house, we create a blueprint.
  Similarly:
  
  class Student {
    String name;
    int age;
   }

   Student is only a blueprint.
   No memory is allocated yet.

2.What is an Object?
  Object = Real thing created from a class.

  Student s1 = new Student();
  Now memory is allocated.
  s1 is an object.

  public class Main {
     public static void main(String[] args) {
        Student s1 = new Student();
        s1.name="Pratish";
        s1.age=21;

        System.out.println(s1.name);
        System.out.println(s1.age);
     }
  }

  class Student {

    String name;
    int age;
  }
